هنا حنبدأ نستعمل ال base 

             System.out.println(base + base + base + base + base + head + head);

لكن حيكون المخرج غير اللي نبغاه، السبب انه base و head كلها ارقام صحيحة، انت بكل بساطة بتجمعها كانها رياضيات ويعطيك ناتج.
0 + 0 + 0 + 1 + 1 = 2
اذا نبغا نجمعها كنص او String لازم نحولها ل String.

            System.out.println("" + base + base + base + base + base + head + head);
