ملاحظة مهمة: اتاكد انك تحذف scnr.close(); او اي كود يقفل السكانر لانه ممكن يسبب ايرور 

        // Output each floating-point value with two digits after the decimal point, which can be achieved as follows:
           System.out.printf("%.2f", yourValue);

ال printf يشبه ال print اللي ما يطبع سطر جديد، لكن ميزته انه يقدر يزين او يسوي format للطباعة.
الطريقة انك تحط علامة النسبة المئوية وبعدها نوع المتغير، مثلا %f وجافا راح تستبدلها بقيمة المتغير. 

            float percentage = 6.641;
            String name = "bro code";
            int count = 5;

            System.out.printf("%s", name); // راح يطبع bro code
            System.out.printf("result is %.1f", percentage); // راح يطبع result is 6.6
            System.out.printf("We completed %.4f of the lab", percentage); // راح يطبع We completed 6.6410 of the lab
            System.out.printf("My name is %s, and I have %03d dollars", name, count) // راح يطبع My name is bro code, and I have 005 dollars

نقدر نحط خصائص لطريقة الطباعة بين ال% والحرف اللي يمثل نوع المتغير، مثلا ال.2 معناتها رقمين بعد الفاصلة فقط.
ونقدر نحط اكثر من متغير في الطباعة، اهم شي انها تكون بنفس الترتيب
