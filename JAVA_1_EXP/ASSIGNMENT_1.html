الميثود هو جزئية من الكود اللي ما يتم تنفيذها مباشرة، لكن يتم تنفيذها اذا تم استدعائها (عكس ال main اللي تتشغل اول ما تشغل البرنامج).
اقرب مثال للميثود هو الفنكشن في الرياضات، مثلا f(x) = x + 1 نقدر نمثلها في جافا: 

            public static Double f(Double x) {

            }

دحين اذا استدعينا الميثود من main، وطبعنا الناتج، مثلا:

الحروف chars في Java
الحرف char نقدر نعينه في جافا باستعمال علامات التنصيص الفردية 

        char harf = 'S';

ونقدر نحوله الى ASCII Code، وهو كود مميز لكل حرف، تقدر تشوف جدول لأهم الحروف واكوادها هنا.
طريقة التحويل باستعمال ال casting، نقدر نحول ال char الى int اللي يمثل ال ASCII Code، والعكس صحيح نقدر نحول int الى الحرف اللي يمثله.
مثال: 

        System.out.println(100); // راح يطبع 100
        System.out.println((char)100); // راح يطبع حرف دي d
        System.out.println("The ASCII Code of G is " + ((int)('G'))); // جربه بنفسك (:
